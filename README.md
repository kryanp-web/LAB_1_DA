# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Bank" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Данные представляют собой транзакции, произошедшие за год в онлайн-компании, не имеющей разничных магазинов. Среди возможных задач ИАД поиск ассоциативных правил, сегментация клиентов, описательная аналитика.
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
